# 기능 목록

- 기초 공사
    - Month: 특정 월을 가져오기 위한 Enum 객체
    - Holidays: 공휴일들을 저장하기 위한 Enum 객체

- 평일 순번 입력
    - [예외]: 근무 순번에 특정 인원이 중복되는 경우
    - [예외]: 평일 비상 근무순번과 주말 비상 근무순번의 인원수가 다른 경우
    - [예외]: 평일 비상 근무순번의 이름과 휴일 비상 근무순번의 사원 이름이 일치하지 않는 경우

    - Member: 각 멤버 이름 기록
    - Members:  멤버 리스트를 저장
    - Schedule: 평일 비상 근무 순번과 주말 비상 근무순번을 저장(Queue)

- 근무표 배정
    - 연속 2일 근무인 경우 다음 근무자와 순서를 바꿈
        - 수아가 평일에 근무를 서고, 다음날 금요일이 휴일이면서 수아가 근무하는 경우 다음 휴일 근무자와 순서를 바꾼다.
        - 휴일에 수아가 근무를 서고, 다음날 평일이 수아인 경우는 다음 평일 근무자와 순서를 바꿔서 근무한다.

- 근무표 출력
    - 비상 근무표를 출력
    - 법정 공휴일에 `(휴일)`표기
